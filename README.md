Kinoplekis - Sistema de Reserva de Ingressos 🎬
(ESTE É UM PROJETO PESSOAL)

Releitura moderna e tecnológica da plataforma de cinema Kinoplex. Este projeto é uma aplicação Full-Stack que permite navegar por um catálogo de filmes, selecionar sessões e reservar assentos em tempo real com persistência de dados.

🚀 Tecnologias Utilizadas
Backend
Python 3.x

FastAPI: Framework web de alta performance para a construção e consumo da API.

SQLAlchemy: ORM para comunicação eficiente e segura com o banco de dados.

SQLite: Banco de dados relacional para persistência do catálogo e das reservas.

Uvicorn: Servidor ASGI para rodar a aplicação.

Frontend
React: Construção da interface do usuário de forma declarativa, modular e baseada em componentes.

TypeScript: Adição de tipagem estática para garantir maior segurança, escalabilidade e previsibilidade no código do lado do cliente.

UI/UX: Layout totalmente responsivo com foco em uma identidade visual premium e moderna (Dark Mode nativo).

✨ Funcionalidades
Catálogo Dinâmico: Os filmes e sessões são consumidos via API e renderizados diretamente a partir do banco de dados.

Mapa de Assentos: Interface interativa e dinâmica para visualização e seleção de poltronas.

Diferenciação VIP: Lógica visual e de sistema estruturada para assentos especiais (padrão KinoEvolution).

Persistência de Dados: As reservas são validadas e salvas no SQLite, impedindo concorrência ou que o mesmo assento seja comprado duas vezes após o recarregamento.

Navegação Dinâmica: Integração fluida entre o catálogo e a tela de checkout via parâmetros de URL e roteamento de componentes.
